# Training Planner

**Автор:** Иван Иванов (Ivan Ivanov)

## Описание

Training Planner - это десктопное приложение для планирования и отслеживания физических тренировок. Программа позволяет:

- Добавлять тренировки с указанием даты, типа и длительности
- Фильтровать тренировки по типу и дате
- Сохранять и загружать данные в формате JSON
- Удалять тренировки через контекстное меню
- Автоматически проверять корректность ввода данных

## Требования

- Python 3.7+
- Стандартные библиотеки: tkinter, json, datetime, os
